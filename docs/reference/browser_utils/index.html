TextRendererResult​

class TextRendererResult()

The result of rendering a webpage to text.

PageTextRenderer​

class PageTextRenderer()

A TextRender is used by the SimpleTextBrowser to claim responsibility for rendering a page. Once a page has been claimed, the instance' render_page function will be called, and the result stream is expected to be consumed -- there is no going back.

claim_responsibility​

def claim_responsibility(url, status_code, content_type, **kwargs) -> bool

Return true only if the text renderer is prepared to claim responsibility for the page.
